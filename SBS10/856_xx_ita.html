Campo 856 - Localizzazione elettronica e accesso (R)







     Facoltativo

     Ripetibile



     Indicatore 1: modalità di accesso



       blank = nessuna indicazione

       1 = FTP

       2 = Telnet

       3 = Dial-up

       4 = HTTP

       7 = Metodo specificato nel sottocampo y





     Indicatore 2: non definito







     Sottocampi:

            $a    Nome dell'host  (R)

            $b    Numero di accesso  (NR)

            $c    Informazioni sulla compressione  (R)

            $d    Path  (R)

            $e    Data dell'ultima consultazione (aaaammggoomm)

            $f    Nome elettronico(R)

            $g    Nome uniforme (R)

            $h    Processore(NR)

            $i    Istruzioni (R)

            $j    Bits per secondo  (NR)

            $k    Password  (NR)

            $l    Logon/login  (NR)

            $m    Contatto per assistanza (R)

            $n    Nome della localizzazione dell'host  (NR)

            $o    Sistema operativo  (NR)

            $p    Porta  (NR)

            $q    Tipo di formato elettronico   (NR)

            $r    Settaggio  (NR)

            $s    Grandezza file  (R)

            $t    Emulatore di terminale  (R)

            $u    URL  (R)

            $v    Orario di accesso  (R)

            $w    Numero di controllo del record  (R)

            $x    Nota non pubblica  (R)

            $y    Molalità di accesso  (NR)

            $z    Nota pubblica (OPAC) (R)
