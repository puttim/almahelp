Campo 106 - Dati codificati: intestazioni usate come soggetto (NR)





     Il campo contiene un codice che indica se l'intestazione può
     essere usata come soggetto.

     Facoltativo
     Non ripetibile

     Indicatori non definiti






     Sottocampi:
     a - Dati codificati (NR)
         0 = può essere usata come intestazione di soggetto
         1 = non può essere usata come intestazione di soggetto
         2 = può essere usata solo come intestazione di soggetto




     Mettere sempre il valore 0
