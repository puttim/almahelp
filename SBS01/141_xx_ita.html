141 - Dati codificati: Caratteristiche specifiche della copia




    Non usato in SBN
    Ripetibile
    Indicatori non definiti




    Sottocampi:
      $a Dati codificati - Caratteristiche specifiche della copia (NR)

            0-2 Materiale della legatura




              a = pergamena
			  b = cuoio
			  c = legno
			  d = tessuto
			  e = materiale sintetico
			  f = cartone
			  g = carta
			  h = non rilegato
			  z = altro
			  [in bianco] = informazione non necessaria o non disponibile




            3 Tipo di legatura




             a = originale
             b = nuovamente rilegato
             c = moderna (dal sec. 20. in poi)
             d = restaurata, facsimile
             e = restaurata, imitazione
             f = esemplare legato con un altro documento(da usare per le legature
                 editoriali, del distributore o del proprietario)
             h = in fogli, non rilegato (senza coperta né cucitura)
             u = sconosciuto
             z = altro (ad es., blocchi cuciti ma privi di coperta)
             [in bianco] = informazione non disponibile




           4 'Legato con'




           Codice di un carattere che indica se la copia descritta è un'unità
		   fisica separata o è legata con uno o più altri documenti.




            0 = non legata con altri documenti
	        1 = legata con uno o più altri documenti




      5  Stato di conservazione - Legatura




            a = eccellente
            b = buono
            c = discreto
            d = danneggiata
            e = rotta
            f = mancante
            g = restaurata
            u = sconosciuto
            z = altro




      6-7 Stato di conservazione - Corpo del libro




            a = eccellente
            b = buono
            c = discreto
            d = danneggiato
            e = incompleto
            g = restaurato
            u = sconosciuto
            z = altro
           [in bianco] = informazione non disponibile





      $b Caratteristiche specifiche della legatura (NR)

      $5  Codice ICCU e sigla della biblioteca (Obbligatorio - NR)


 

		  Codice e sigla della biblioteca che possiede la copia cui si riferisce
		  il campo (utilizzare il comando Ctrl+F8 e selezionare il codice dalla
		  lista); se la singola biblioteca possiede altre copie dello stesso
    	  documento, inserire anche la collocazione preceduta da ": " (due punti,
	      spazio)





   N.B.:
        - Per opere in più volumi descritti in un unico record compilare
          un solo campo 141, descrivendo lo stato di conservazione del volume in
          peggiori condizioni. Lo stato di conservazione dei singoli volumi viene
          descritto nella nota 316.
         - Per compilare il $5 posizionarsi sul sottocampo, inserendo uno spazio si apre una lista
      da cui selezionare il codice appropriato
