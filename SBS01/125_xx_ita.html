      $a Formato della musica a stampa (NR)
      Posizione 0 - Tipo di partitura




            Codice di un carattere, indica il tipo di partitura. [codici non usati in SBN]
              a = partitura completa, parte con guida
              b = partiturina, partitura tascabile, partitura da studio
              c = spartito, con accompagnamento ridotto per strumento a tastiera
              d = partitura vocale, partitura per coro, senza accompagnamento
              e = partitura condensata, parte di pianoforte [violino ecc.] conduttore
              f = partitura grafica
              g = partitura ristretta (es. innarii)
              h = intavolatura
              i = libro corale
              j = particella
              k = parte solista (cartina)
              m = più formati (indicare i singoli formati nel sottocampo $c)
              n = spartitino
              o = testo e accordi
              p = partitura da tavolo
              u = sconosciuto
              x = non applicabile, non è una partitura
              z = altro




      	Posizione 1 - Indicatore delle parti [non usata in SBN]




              Codice di un carattere, indica l'esistenza di parti per una partitura.
              a = le parti esistono
              b = parti strumentali
              c = parti vocali
              u = sconosciuto
              x = non applicabile
              y = parti non presenti
