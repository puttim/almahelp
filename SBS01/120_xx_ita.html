120 - Dati codificati: Materiale cartografico




    Per SBN: compilabile solo per Tipo materiale = C (Cartografia)
    Obbligatorio
    Non ripetibile
    Indicatori non definiti





    Sottocampi:
      $a Materiale cartografico, dati codificati (Generale)
      0 Indicatore del colore




              a = un colore
              b = multicolore





     Per SBS: compilare almeno la pos. 0





      1 Indicatore dell'indice




              a = indice o lista sul medesimo materiale cartografico
              b = indice o lista acclusa al materiale cartografico in un libretto, pamphlet, coperta staccata, etc.
              c = indice o lista presenti ma posizione non specificata
              y = nessun indice o lista





      2 Indicatore per il testo esplicativo




              a = testo sul medesimo materiale cartografico
              b = testo accluso al materiale cartografico in un pamphlet,
                   coperta, etc.
              y = nessun testo esplicativo





      3-6 Codice del campo per i rilievi
